## 2. 阶段开发内容与进度详情

### 第一阶段：基础架构与数据闭环 (已完成)
- **技术栈选型**：Next.js 15+ (App Router) + Payload CMS 3.x + PostgreSQL 17 + Docker Compose。
- **CMS 数据模型（Collections）定义**：
  - `Admins` (后台管理员)
  - `Clients` (前台 B2B 客户)
  - `Dramas` (短剧作品，包含题材、语言、授权市场、适配平台、推荐位、版权管理等)
  - `Episodes` (分集信息，包含集号唯一性验证、关联视频资产)
  - `VideoAssets` (视频资产，存储对象包括 ObjectKey、视频格式、转码状态、时长等)
  - `Media` (图片、文档等媒体库)
- **B2B 鉴权与防盗链签名 API**：
  - 编写了 `/api/video-assets/sign-play-url` 接口，根据客户端 `episodeId` 进行客户鉴权（校验 active 状态），并返回有时效限制 of 带签名临时播放 URL（不暴露原始 OSS 路径）。
- **验证**：编写并顺利通过了集成测试 `dataClosedLoop.int.spec.ts`。

---

### 第二阶段：前端核心页面开发与响应式适配 (已完成)
- **多语言前缀路由重构**：将前端目录重构为 `/[locale]` 结构，支持 `/zh`、`/en`、`/ar`、`/tr`，并将根路径 `/` 默认重定向至 `/en`。
- **RTL 阿拉伯语自适应**：在局部多语言 Layout 中，为 `ar` 语种动态注入 `dir="rtl"`，使用 CSS 逻辑属性（Logical Properties）实现排版、网格、边距的完美镜像。
- **前台核心页面开发**：
  - **首页**：全屏龙标影视主视觉、Highlights 亮点数据、4 大业务卡片展示、精选短剧列表、合作方 Logo 墙。
  - **关于我们**、**业务服务**、**新闻动态列表及详情页**（自主开发 Lexical 富文本渲染器）。
  - **联系我们**：包含 B2B 商务表单（提交写入 `inquiries` 表）以及科技感 CSS 经纬度雷达地图。
  - **作品库**：实现包含题材、语言、授权市场、流媒体适配渠道、集数范围等在内的 **六维复合筛选系统**。
- **登录流程简化**：根据用户建议，将前台登录重构为直接使用 **“电子邮箱 + 密码”** 方式登录，去除了多余的用户名到邮箱转化层。
- **验证**：通过了所有 Playwright 端到端（E2E）测试 `tests/e2e/`。

---

### 第三阶段：视频播放与权限控制、登录流程 (已完成)
- **专业流媒体播放器集成**：使用 `video.js` 替换原生的 HTML5 `<video>` 标签，支持标准 `MP4` 视频以及自适应 `HLS` (`m3u8`) 切片流的流畅播放。
- **前端登录拦截**：游客在短剧详情页点击分集列表中的任意集数或大播放按钮时，在当前页面弹出悬浮的毛玻璃登录 Modal。
- **无刷新继续播放**：客户输入邮箱密码成功登录后，弹窗自动关闭，播放器立刻在后台拉取临时签名 URL 并直接开始加载播放。
- **视频安全防盗**：
  - 禁用鼠标右键菜单，隐藏原生控制栏的“下载”与画中画按钮。
  - **防录屏水印**：在播放器上叠加一层包含当前登录客户的 `Email` 和实时更新时间戳的半透明水印，每 8 秒自动在屏幕边缘随机位移，从源头上预防使用录屏软件盗取短剧内容的行为。

---

### 第四阶段：CMS 后台功能定制与动态合作方 Logo 墙 (已完成)
- **CMS 动态合作方管理**：配置了 `Partners` 关系表，支持管理员在后台动态新增、修改、控制排序及启用/禁用合作伙伴 Logo 墙数据。
- **多语言翻译状态管理**：在 `Dramas` 表中增加了翻译状态组（包括简体中文、阿拉伯语、土耳其语的翻译阶段跟踪：Pending/Translated/Reviewed）。
- **游客与客户数据过滤**：在 `Episodes` 和 `Dramas` 的 Access 控制中限制游客与前台客户只能查询 `enabled: true` 以及 `_status: 'published'` 的剧集资产。

---

### 第五阶段：PWA 应用支持、SEO 与社交分享优化 (已完成)
- **PWA 渐进式 Web 应用配置**：
  - 在 `public/` 目录下新增并配置了 `manifest.json`，声明应用主题色与独立窗口启动模式。
  - 配置了金龙图标规格（`icon-192.png` 和 `icon-512.png`）作为桌面快捷安装图标。
  - 并在前端顶层 `layout.tsx` 中注册声明，实现了移动端可保存至桌面的 App 化体验。
- **多语言高级 SEO**：
  - **短剧详情页**：动态注入了 **Schema.org 的 `Movie` 和 `VideoObject` 结构化数据**，并优化了多语言 `Open Graph` 与 Twitter Sharing 元标签。
  - **新闻详情页**：动态注入了 **`NewsArticle` 结构化 JSON-LD 数据**。
- **海外社媒一键分享**：在短剧详情“版权与授权信息”卡片下方，设计了符合多语言预设文案的 **“WhatsApp 一键分享”按钮**，拉起客户端推荐给合作伙伴。
- **性能与类型校验**：修复了编译过程中所有与 Payload 联表查询相关的 TypeScript 类型约束及 ESLint 语法警告，打包成功率为 100%。
