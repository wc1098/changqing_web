## 1. 项目概述

### 1.1 公司信息

| 项目 | 内容 |
|------|------|
| 公司名称 | 东方视野影业 |
| 英文全称 | Eastern Vision Media & IP Management FZCO |
| 总部所在地 | 迪拜（Dubai） |
| 市场定位 | 中东及北非（MENA）市场短剧工业化全链路服务商与发行商 |
| 核心业务 | 中国精品短剧出海 |
| 企业使命 | 让优质短剧实现本地化落地与商业变现 |

### 1.2 成立背景

依托迪拜优厚政策，拥抱中东与北非短剧市场发展新机遇。

### 1.3 主要业务

1. **短剧全链条制作**：涵盖剧本创作直至成片产出
2. **中东市场发行与渠道对接**
3. **品牌推广与本地化营销**
4. **IP 孵化与内容定制**

### 1.4 驱动模式

以"优质内容输出 + 本地化深度适配"为双轮驱动，构建从精品短剧版权供给、定制化联合制作到本地化运营发行的完整闭环。

## 2. 项目定位

### 2.1 核心定位

**纯 B2B 短剧内容展示平台**

- 核心目标：向中东及北非地区的流媒体平台、制作方、投资方等 B 端客户系统化展示东方视野影业自有短剧资源
- 不包含任何推广分佣、收益分成、用户导流等商业化变现路径
- 所有内容展示均以客观、完整、可验证为原则，服务于 B 端客户的选片与合作评估

### 2.2 目标用户

| 用户类型 | 说明 |
|---------|------|
| 中东本地流媒体平台/渠道方 | 寻找优质短剧内容采购 |
| 中国短剧制作方 | 寻找海外发行渠道 |
| 投资方/合作伙伴 | 评估公司实力与合作价值 |

> 注：不面向普通观众用户

### 2.3 网站目的

- **品牌展示**：建立专业、可信赖的企业形象
- **业务获客**：吸引 B 端客户主动联系合作
- **内容展示**：系统化展示短剧内容资产

## 3. 品牌视觉规范

### 3.1 Logo

- 图案：金龙 + 菱形组合
- 色彩：金色渐变
- 提供格式：矢量图 + PNG（横版/竖版）

### 3.2 主色调

- **主色**：金色（Gold）
- **辅助色**：待设计确认
- **背景色**：深色主题（契合影视行业调性）

### 3.3 设计调性

- **东西方融合的 Cultural Bridge 感**
- 高端商务 + 影视娱乐的平衡
- 适度动效，不夸张

### 3.4 多语言与布局

- 支持语言：中文、英文、阿拉伯语、土耳其语
- 阿拉伯语需支持 **RTL（从右到左）布局**

### 4.3 作品库页面

#### 4.3.1 展示方式

- **卡片网格布局**
- 每张卡片包含：竖版海报 + 剧名 + 分类标签 + 语言标签
- 响应式列数：PC 端 4-5 列，平板 2-3 列，手机 1-2 列

#### 4.3.2 六维筛选系统

| 筛选维度 | 说明 | 示例 |
|---------|------|------|
| 题材分类 | 短剧类型 | 总裁/复仇/浪漫/悬疑/古装... |
| 语言版本 | 配音/字幕语言 | 英语/阿拉伯语/土耳其语... |
| 集数范围 | 剧集数量区间 | 1-10集/11-30集/30集以上 |
| 上线时间 | 发布时间排序 | 最新上架/按年份 |
| 地区/市场 | 目标市场 | 沙特/阿联酋/土耳其... |
| 合作平台 | 已分发平台 | 各合作平台名称 |

#### 4.3.3 数据规模

- 作品总数：约 **1000 部**
- 更新频率：不定期随机更新
- 需要分页加载和懒加载优化

## 5. 访问权限模型

### 5.1 权限规则

| 功能 | 未登录访客 | 已登录用户 |
|------|-----------|-----------|
| 浏览首页/关于/业务/新闻 | ✅ 允许 | ✅ 允许 |
| 浏览作品列表 | ✅ 允许 | ✅ 允许 |
| 查看作品详情（文字/图片） | ✅ 允许 | ✅ 允许 |
| **播放视频（预告片/分集）** |  拦截 | ✅ 允许 |

### 5.2 登录触发机制

- **触发时机**：用户在作品详情页点击播放按钮时
- **拦截方式**：弹出登录弹窗（不跳转页面）
- **登录后行为**：关闭弹窗，继续播放视频

### 5.3 账号管理

- **注册方式**：不支持前端自助注册
- **账号来源**：由后台管理员创建并分发
- **账号格式**：用户名 + 密码
- **账号状态**：可启用/禁用

## 6. CMS 后台功能

### 6.1 功能模块清单

| 序号 | 模块 | 功能说明 |
|------|------|---------|
| 1 | 作品库管理 | 短剧的增删改查、海报上传、简介编辑、分类关联、预告片/分集视频上传 |
| 2 | 页面内容管理 | 全站页面文案可后台编辑（关于我们/业务服务/联系我们等） |
| 3 | 新闻/动态管理 | 新闻的增删改查、分类管理、封面图上传 |
| 4 | 合作方管理 | 合作方 Logo 上传、排序、启用/禁用 |
| 5 | 多语言管理 | 每条内容支持中/英/阿/土四语编辑 |
| 6 | 用户账号管理 | 创建/编辑/禁用/删除登录账号 |
| 7 | 商务表单管理 | 查看访客提交的商务合作留言 |
| 8 | 媒体资源库 | 统一管理图片、视频等素材资源 |
| 9 | 分类/标签管理 | 题材/语言/地区/集数等维度的增删改 |
| 10 | SEO 配置 | meta 标签、结构化数据、OG 标签配置 |
| 11 | 基础设置 | 网站名称、Logo、联系方式、社交媒体链接等 |
| 12 | 首页推荐位管理 | 手动指定首页展示的精选作品 |

### 6.2 后台界面风格

- **现代化 Dashboard 风格**（类似 Notion/飞书）
- 简洁、专业、易用

### 6.3 多语言内容管理策略

- 每种语言的内容由人工翻译后手动填入
- 后台提供翻译状态追踪（已翻译/待翻译）
- 默认语言：英文（其他语言未填写时回退到英文）

## 7. 技术要求

### 7.1 响应式设计

| 设备类型 | 屏幕宽度 | 布局特点 |
|---------|---------|---------|
| 手机 | < 768px | 单列/双列卡片，上下布局播放器 |
| 平板 | 768px - 1024px | 2-3 列卡片，适配触摸操作 |
| PC | > 1024px | 4-5 列卡片，左右布局播放器 |

### 7.2 多语言支持

- 语言切换器：Header 导航栏显示
- URL 结构：`/zh/...`、`/en/...`、`/ar/...`、`/tr/...`
- 阿拉伯语自动切换 RTL 布局
- hreflang 标签支持（SEO）

### 7.3 动效设计

| 动效类型 | 位置 | 说明 |
|---------|------|------|
| Logo 光效 | 首页主视觉 | 金色光效流动，适度不夸张 |
| 数字滚动 | 数据亮点模块 | 从 0 滚动到目标数字 |
| 卡片悬停 | 作品卡片 | 轻微放大/显示简介 |
| 滚动渐入 | 全页面 | 元素进入视口时淡入 |

### 7.4 PWA 支持

- 可添加到手机桌面（像 App 一样使用）
- 离线缓存基础页面
- 推送通知（后续迭代）

### 7.5 社交分享

- WhatsApp 分享按钮（中东地区普及率高）
- OG 标签自定义分享卡片

### 7.6 视频播放

- 播放器：xgplayer 或 video.js
- 格式支持：MP4 / HLS
- 移动端：全屏自动横屏
- 防盗措施：禁用右键/下载按钮（基础防护）

### 7.7 SEO 优化

- 多语言 meta 标签（title/description/keywords）
- 结构化数据（Schema.org - Organization/VideoObject）
- OG 标签（社交媒体分享卡片）
- 多语言 sitemap
- 语义化 HTML 结构

## 8. 基础设施

### 8.1 云平台

- **服务商**：阿里云国际版
- **理由**：中东地区节点覆盖好，符合公司迪拜总部定位

### 8.2 存储与分发

| 资源类型 | 存储方案 | 分发方案 |
|---------|---------|---------|
| 图片（海报/Logo/剧照） | OSS | CDN 全球加速 |
| 视频（预告片/分集） | OSS | CDN 全球加速 |
| 静态资源（JS/CSS/字体） | 服务器本地 | CDN 加速 |

### 8.3 视频安全

- **OSS 防盗链**：Referer 白名单 + URL 签名
- **临时播放地址**：后端鉴权后生成带时效的 URL
- **可选水印**：后续可叠加用户账号水印（防录屏）

### 8.4 域名

- 域名由用户后续自行解析
- 需配置 SSL 证书（HTTPS）

## 9. 数据规模与性能

### 9.1 数据规模预估

| 数据类型 | 数量 | 说明 |
|---------|------|------|
| 短剧作品 | ~1000 部 | 持续增加 |
| 每部剧集数 | 平均 20-80 集 | 差异较大 |
| 合作方 | ~10 家 | 相对稳定 |
| 登录用户 | 待确认 | B 端客户账号 |
| 新闻文章 | 待确认 | 不定期发布 |

### 9.2 性能要求

- 首页加载时间：< 3 秒（中东地区网络）
- 作品列表加载：< 2 秒（分页/懒加载）
- 视频起播时间：< 3 秒（CDN 加速）
- 移动端适配：所有主流机型

## 11. 后续迭代方向

上线后根据业务需求逐步迭代：

- [ ] 观看数据分析（哪些剧被播放最多、用户行为追踪）
- [ ] 账号分级权限（不同客户看不同内容）
- [ ] 视频水印（显示登录账号信息，防录屏）
- [ ] 账号有效期管理（试用账号自动过期）
- [ ] 多语言翻译工作流（协作翻译流程）
- [ ] 高级搜索（全文检索）
- [ ] 数据看板（访客统计/热门内容/转化分析）
- [ ] 推送通知（新剧上线通知）

## 12. 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 四语翻译质量 | 用户体验差 | 专业翻译 + 人工校对，不依赖机器翻译 |
| 视频播放卡顿（中东网络） | 用户流失 | CDN 多节点 + HLS 自适应码率 |
| 1000 部作品数据导入 | 上线延迟 | 批量导入脚本 + 分批次上线 |
| RTL 布局兼容性问题 | 阿拉伯语体验差 | 早期介入测试，逐页面验证 |
| 开发周期紧张 | 功能不完整 | 功能优先级排序，MVP 先行 |
